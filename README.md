## ✨ 项目简介

幻境 PPT 助手是一个无需安装、开箱即用的 AI 驱动演示文稿生成工具。只需输入主题，AI 将自动生成完整的 PPT 大纲和内容，支持 10 种精美主题和 6 种专业排版布局。

### 🎯 核心亮点

- 🤖 **AI 驱动内容生成** - 自动生成结构化的幻灯片内容，包含数据、案例和要点
- 🎨 **10 种专业主题** - 从商务到创意，满足各种场景需求
- 📐 **6 种排版布局** - 经典、大数据、时间线、对比、图表、图片网格
- 🚀 **实时预览** - 所见即所得，预览样式与导出 PPT 完全一致
- 💫 **酷炫视差动画** - 四层视差滚动 + 粒子效果 + 发光装饰
- 📦 **一键导出** - 导出为可编辑的 .pptx 文件

## 🛠️ 技术栈

- **前端**: Vue 3 + Tailwind CSS
- **PPT 生成**: PptxGenJS
- **AI 服务**: OpenAI API (可配置模型)
- **模块化**: ES Modules
- **图标**: Lucide Icons

## 🚀 快速开始

### 前置要求

- 一个支持 ES Modules 的现代浏览器（Chrome 61+, Firefox 60+, Safari 11+）
- OpenAI API 密钥（或兼容的 API 服务）

### 本地运行

**重要**: 由于使用了 ES Modules，必须通过本地服务器运行。

**方法 1: Python**
```bash
python -m http.server 8080
# 访问 http://localhost:8080
```

**方法 2: Node.js**
```bash
npx http-server -p 8080
```

**方法 3: VS Code Live Server**
- 安装 Live Server 扩展
- 右键 `index.html` → "Open with Live Server"


### 使用步骤

1. **配置 API**
   - 输入 OpenAI API 密钥（或兼容服务的 API 地址）
   - 可自定义文字模型（默认 gpt-5.2）和图像模型（默认 gpt-image-1）

2. **输入主题**
   - 输入 PPT 主题和补充要求
   - AI 自动生成幻灯片大纲

3. **编辑大纲**
   - 可增删改大纲中的幻灯片
   - 调整标题和描述

4. **选择主题**
   - 从 10 种视觉主题中选择
   - 可上传自定义背景图

5. **生成内容**
   - AI 逐页生成详细内容
   - 实时预览，支持编辑和重新生成

6. **导出 PPT**
   - 一键导出为 .pptx 文件
   - 可在 PowerPoint/WPS 中继续编辑

## 🎨 功能特性

### 视觉效果
- **四层视差滚动** - 背景粒子、光晕、装饰线条、主内容以不同速度移动
- **粒子爆发** - 步骤切换时触发亮度闪烁和模糊效果
- **波纹扩散** - 光晕从中心向外扩散
- **脉冲发光** - 6 条装饰线条持续呼吸发光
- **启动仪式** - Step 4→5 特殊过渡动画
- **动画开关** - 右上角一键开关所有动画

### PPT 主题 (10 种)
商务精英 | 赛博幻境 | 极简主义 | 灵感极光 | 科技未来 | 学术论文 | 自然清新 | 复古怀旧 | 渐变梦幻 | 深蓝商务

### 排版布局 (6 种)
- **经典布局** - 文本 + 列表 + 图片
- **大数据布局** - 突出数字展示
- **时间线布局** - 流程步骤
- **对比布局** - 左右两列对比
- **图片网格** - 2×2 图片展示
- **图表布局** - 数据可视化（柱状图/饼图/折线图/面积图）

### 内容生成
- **逐页生成** - 避免批量失败，实时更新
- **结构化内容** - 包含具体数据、百分比、案例
- **智能布局** - AI 自动选择最合适的布局类型
- **可编辑** - 支持手动编辑或 AI 重新生成

### 预览功能
- **双栏布局** - 左侧缩略图 + 右侧大预览
- **固定比例** - 16:9 比例，与导出 PPT 一致
- **所见即所得** - 字体、颜色、排版完全对应

## 🔧 配置说明

应用配置（API 密钥、模型等）会自动保存到浏览器 LocalStorage，无需每次输入。

### 支持的 API 服务
- OpenAI API
- 任何兼容 OpenAI 格式的 API 服务

### 模型配置
- **文字模型**: 用于生成大纲和内容（默认 gpt-5.2）
- **图像模型**: 用于生成配图（默认 gpt-image-1）
